# Sanraw - Integrated Business Management System

Sanraw is a comprehensive full-stack web application designed to streamline business operations for paddy and rice-related enterprises. It provides a robust platform for managing inventory, tracking sales, generating bills, and monitoring financial reports.



## Key Features

**Authentication & Security:** Secure login system with role-based access control and JWT-based session management.
**Dynamic Dashboard:** Visualized business data using interactive charts to monitor sales growth and stock levels.
**Sales & Billing:** Efficient point-of-sale interface to generate bills for rice, paddy, equipment, and chemicals.
**Inventory Management:** Real-time tracking of stock levels with automated updates upon sales and purchases.
**Supplier & Purchase Tracking:** Manage supplier information and track raw material procurement.
**Credit Management:** Track customer credit history and payment status efficiently.
**Comprehensive Reporting:** Generate daily, monthly, and category-wise reports for better decision-making.

## Tech Stack

### Backend
* **Node.js & Express:** Scalable server-side logic.
* **MySQL:** Relational database for structured data management.
* **JWT & Bcrypt:** Secure authentication and password hashing.
* **Cloudinary:** Cloud storage for product and profile images.
* **Nodemailer:** Automated email notifications.

### Frontend
* **React.js:** Modern UI component architecture.
* **Material UI (MUI):** Professional design and responsive layouts.
* **Recharts:** Dynamic data visualization.
* **React Context API:** Efficient global state management.

## Installation & Setup

### Prerequisites
* Node.js (v14 or higher)
* MySQL Server
